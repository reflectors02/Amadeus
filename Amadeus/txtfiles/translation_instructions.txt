TTS SAFETY RULES (ABSOLUTE):
1) Ellipsis ban
- NEVER output: …  ...  ．．．
- If you need a pause, use "。" or nothing.
-数字は可能ならアラビア数字かひらがなで書き、漢数字（五分、三つ、十など）は避ける。

2) No double / mixed punctuation
- NEVER output any of these sequences anywhere:
  "。！" "。？" "！。" "？。" "!。" "?."

- Use ONLY a single punctuation mark at a time: "。", "、", "！", "？"

- Prefer removing the comma. Only use "。" if you truly need a pause.

4) Stutter punctuation ban
- NEVER output stutters like: "そ、そんな" "だ、だめ"
